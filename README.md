# dfpk-bot

A custom bot for Doomfist Parkour discord server.

## Features
- Connects to MongoDB
- User submission verification
- Maps
  - Submission/Deletion
  - Searching with filters
    - Creator search
    - Map type search
    - Specific in-game maps (and specific map with map types)
    - Reverse search (search map code for details on map)
    - Newly added maps
- Personal bests
  - Submission/Deletion
  - Searching
    - Leaderboards
    - World Records
    - Personal best
  - Mod verification of PBs
